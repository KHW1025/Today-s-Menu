<br>

> **[오늘의 메뉴 페이지](https://port-0-today-s-menu-1msx2blg1sfry0.sel3.cloudtype.app/)**
<br>


## 1️⃣ 목적 & 구성
<br>

> - 요리 레시피들을 공유하고 mongodb를 활용해 추가, 수정, 삭제가 가능하도록 하였습니다.
> - nodejs 환경에서 ejs 문법을 사용하였습니다.
> - 미니 기능으로 주사위 버튼을 누르면 랜덤으로 추천하는 기능도 적용시겼습니다.
<br>
